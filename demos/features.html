Layers CSS reference

Here's a clean CSS package that makes it easy to pick the stuff that you need, without having to override things. It doesn't choose a visual style or color scheme for you, it just gives you tools. Native input elements preserved, no dependencies or requirements. (Non-semantic class names: they work, deal with it.)

Please note that the color scheme is on this page is included in extras, the main release doesn't include any colors. Take a look at the source code to see what's added.

Look here to see how native elements look with nothing but Layers CSS loaded.

Fluid grid

Columns within columns work fine. No wrappers are required. You need to mark the last column of the row, and using .clear after rows is a good idea, too.

.column.eight

.column.four last

You can .push- columns.

.column.four

.column.three.push-two

.column.two.last

If you want to start the stack from the .right:

.column.right.five

.column.right.two.push-two

.column.right.three

Without gutters

.column.six.reset

.column.three.reset

.column.three.reset

You can also use proportional keywords:

.column.half.first

.column.fourth

.column.fourth

Fixed-fluid column combination.

One part gets a fixed width, the other side fills the rest. Very handy for combining images with a fluid grid. You can adjust the size of the fixed-width in steps with media queries.

.column.fluid > .column-container

.column.fixed

You can stack this from the .right, too.

.column.fluid.right > .column-container

.column.fixed.right

Custom fixed width/height added.

Common tools

You can automatically use these as mixins in LESS.

.reset
Reset margins to 0
.squeeze
Use smaller margins (half of the default)
.buffer
Add comfortable paddings (like in this box)

Directions work as modifiers:

  • .reset-top
  • .squeeze-right
  • .buffer-left.buffer-right
.clear
Clear stuff
clear: both;
.hidden
Hide stuff
display: none;
.dry
Move inline content where only Hubble can see it (handy for creating buttons with the text hidden)
.reset
Remove margins, paddings, borders etc.
.border-box
